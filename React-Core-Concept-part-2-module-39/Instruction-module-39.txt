সত্যিকারে চলে এলো, লাজুক বাবু React !!! (রিলিজ হচ্ছে মডিউল 39)

#Module_release #Simple_react_milestone #Module_39

প্রেম একদিনে ধরা দিলে তো প্রেম হবে না। তাই আজকে প্রেমের একটু ছিঁটেফোঁটা দিবে React । একটু আনাঘোনা করবে। হালকা কিছু জিনিস বলবে। খুব বেশি না। প্রথম প্রেমে হালকা লাজুক ভঙ্গিতেই থাকবে। তোমারও পুরাপুরি মোহ আজকে ভাঙ্গবে না। কিছুটা বুঝবে আর কিছুটা বুঝবে না। তবে সেটা নিয়ে টেনশন করার কিছু নাই। React তোমার কাছে যেহেতু আসতে শুরু করেছে। সে ভালোভাবেই আসবে। একটু ধীরে ধীরে। 





আজকের মডিউল:

খুবই সহজ একটা মডিউল । সিম্পল কয়েকটা কোর কনসেপ্ট সম্পর্কে জানবে। React সম্পর্কে জানবে। React দিয়ে ভালো কোন এপ্লিকেশন বানানোর মেইন ফাউন্ডেশন হচ্ছে এই কয়েকটা কনসেপ্ট। মোট ৯টা ভিডিও আছে। ভিডিও ৯ টা হলেও দেখবে,বোঝার দরকার নাই। জাস্ট চোখ-কান খুলে দেখবে। প্লাস এই ভিডিওগুলার মেইন অংশ আগামীকাল আবার রিক্যাপ (রিভিশন) দেয়া হবে। সো, আজকে পুরোপুরি বুঝতে না পারলেও টেনশন করার কিছু নাই। আগামীকাল আবার কথা হবে। প্লাস আগামীকাল কন্সেপচুয়াল সেশন আসবে। আগামীকাল প্রাকটিস ডে আসবে। 



এই ৯টা ভিডিও এর মধ্যে একদম শুরুতে আছে কিভাবে React ইন্সটল করবে। কিভাবে React App ইউজ করবে। যদিও এই জিনিসগুলা আগে থেকেই জানো। তারপরেও ধারণা দেয়া হলো। একটা React App এর কোড কিভাবে রাখে, কিভাবে অর্গানাইজ করে সেটা নিয়ে প্রাথমিক আলোচনা। এরপর শুরু কথা হবে Event নিয়ে। এতদিন Js দিয়ে ইভেন্ট হ্যান্ডেল করেছো। আজকে React এ Event -Handling এর ব্যাসিক দেখবে। 

আজকে আরেকটা Interesting টপিক নিয়ে আলোচনা করেছি। সেটা হলে আংটা। মানে হচ্ছে Hooks। আরও আছে State । কোন একটা জিনিস যদি ইউজার চেইঞ্জ করে সেটা কিভাবে ওয়েবসাইট এ দেখাবে। এইটা কি এবং কেনো দরকার সেটা নিয়ে কথা বলেছি। না বুঝলেও সমস্যা নাই। পরের মডিউলগুলো দেখলে এবং প্র্যাক্টিস করলে এগুলো আয়ত্তে আসবে। সেসাথে একটা Advance টপিক ইফেক্ট/UseEffect() হুক নিয়ে কথা বলেছি। এইটা দিয়ে আমরা কিছু Api থেকে ডাটা Fetch করার কাজ করেছি। কিভাবে React এর মধ্যে তুমি API কল করে ডাইনামিক ডাটা এনে খুব সহজেই সেই ডাটা দেখিয়ে দিতে পারো। যদিও Data ফেচিং এর জন্য সামনে আমরা আরও ভিন্ন এবং ভালো কিছু জিনিস শিখবো।



তারপর আমরা Overall একটা পেইজকে কিভাবে আমরা রিয়েক্টের Way তে চিন্তা করতে পারি এবং সেটাকে কিভাবে ছোট কম্পনেন্টে ভেঙ্গে UI বানাতে পারি সেটা নিয়ে আলোচনা করছি। এই জিনিস তোমাকে সামনে প্রজেক্ট করতে সাহায্য করবে। একদিনে কিন্তু এই চিন্তা তৈরি হবে না। আজকেই যে সব শিখে ফাটিয়ে দিবে তেমনটাও না। তবে চিন্তা করার উপায় যেনে রাখো। সামনে প্রজেক্ট করতে করতে আস্তে ধীরে এই থিঙ্কিং ইম্প্রুভ হবে।

সব শেষে রিয়েক্টের ডকুমেন্টেশন নিয়ে কথা বলেছি। এইটা তোমার বন্ধু হবে রিয়েক্ট শিখার জার্নি তে। কারণ, যত বেশি পড়বে ততোই তোমার Learning ভালো হবে। 





React বেসিক Recap

আজকে রিলাক্স মুডে জিনিসগুলা দেখো। যতটুকু পারো বুঝার চেষ্টা করো। বেশ কয়েকটা জিনিসে কনফিউশন থেকে যাবে। সেটা নিয়ে টেনশন নিও না। আগামীকালকের সেইম জিনিসগুলো আরেকবার বুঝিয়ে দেয়া হবে প্লাস React কিভাবে কাজ করে সেটা নিয়ে ধারণা দেয়া হবে Conceptual Session এ। সো, আজকে যদি অল্পও বুঝো তাহলে ভিডিওগুলো একাধিকবার দেখো, বুঝলেও একাধিকবার দেখো,না বুঝলেও একাধিকবার দেখো। সাপোর্ট সেশনে যাও। নো মডিউল ডে- তে কিছু জিনিস Practice করে ফেলো।



ভালোমানের একজন React ওয়েব ডেভেলপার হওয়ার জন্য তুমি যতক্ষণ লেগে থাকবে, একই জিনিস ঘুরিয়ে পেঁচিয়ে বিভিন্নভাবে দেখার চেষ্টা করবে। যত হার্ডওয়ার্ক করবে; সামনের জিনিসগুলো করা তত সহজ হয়ে যাবে।
